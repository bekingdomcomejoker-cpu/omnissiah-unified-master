## Getting Started

### Clone the Unified Repository

```bash
git clone https://github.com/bekingdomcomejoker-cpu/omnissiah-unified-master.git
cd omnissiah-unified-master
```

### Install Dependencies

```bash
pnpm install
```

### Run the Engine

**Option 1: Python Core Only (Fastest)**
```bash
python3 omnissiah_engine.py
```

**Option 2: React Dashboard + Dev Server**
```bash
pnpm dev
```

**Option 3: Static Build + Preview**
```bash
pnpm build
python3 -m http.server 8080 --directory dist/public
```

### Test on Termux/Mobile

See `TERMUX_WORKFLOW.md` for mobile-specific instructions.
